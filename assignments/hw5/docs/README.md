# 作業 5：網路相簿 (圖片處理)

## 任務目標

建立個人相簿，支援批次上傳與縮圖產生。

## 功能需求

1. **圖片上傳**：支援多個圖片檔案上傳
2. **縮圖產生**：使用 PHP GD 自動產生縮圖
3. **圖片管理**：顯示相簿中的所有圖片
4. **刪除功能**：移除不需要的圖片

## 技術重點

- **檔案驗證**：檢查檔案大小、MIME 類型
- **縮圖產生**：使用 `imagecopyresampled()` 產生縮小版本
- **資料庫設計**：圖片資訊存儲

## 評分標準

- 功能完整性 (40%)
- 圖片處理品質 (30%)
- 程式碼結構 (20%)
- 使用者介面 (10%)
