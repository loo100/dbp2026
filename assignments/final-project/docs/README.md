# 期末專案：電子商務網站開發

## 專案概述

整合本學期所學技術，開發一個功能完整的電子商務平台。

## 核心功能模組

### 1. 會員模組
- 使用者註冊與登入
- 個人資料修改
- 訂單查詢與歷史記錄
- 密碼重設功能

### 2. 商品模組
- 商品分類瀏覽
- 關鍵字搜尋
- 商品詳情頁面
- 商品評論與評分

### 3. 購物模組
- 購物車管理
- 結帳流程（模擬）
- 訂單生成
- 訂單確認

### 4. 管理後台
- 商品上架/下架
- 訂單狀態管理
- 銷售統計圖表
- 使用者管理

### 5. 進階技術
- 使用 **Ajax** 實現無重新整理的購物車更新
- 使用 **PHP GD** 產生訂單報表
- 整合 **Laravel** (可選) 進行部分模組重構

## 技術要求

### 必須使用的技術
- HTML5、CSS3、JavaScript
- PHP 7.4+
- MySQL 5.7+
- PDO 資料庫連線
- Session 與 Cookie
- Ajax/Fetch API

### 可選的進階技術
- Laravel 框架
- PHP GD 繪圖
- PHPMailer 郵件功能
- 支付模擬（不需要真實支付）

## 資料庫設計

### 核心資料表
- `users` - 使用者表
- `categories` - 商品分類表
- `products` - 商品表
- `orders` - 訂單表
- `order_items` - 訂單項目表
- `reviews` - 評論表

3. **第 3 週**：商品模組開發

## 常見問題

**Q: 是否需要真實的支付功能？**
A: 不需要，可以模擬訂單生成即可。

**Q: 是否必須使用 Laravel？**
A: 不必須，但使用 Laravel 可獲得額外加分。

**Q: 資料庫有最低要求嗎？**
A: 至少需要 5 個資料表，建議設計正規化的資料庫。
