# 作業 3：投票系統 (PHP GD 繪圖)

## 任務目標

建立一個線上投票系統，並將結果以圖表形式呈現。

## 功能需求

1. **發起投票**：管理者可設定投票題目與選項
2. **進行投票**：使用者選擇選項並送出
3. **結果顯示**：使用 PHP GD 函式庫動態產生長條圖或圓餅圖

## 技術重點

- **PHP GD 基本函數**：`imagecreate()`, `imagecolorallocate()`, `imagestring()`, `imagepng()`
- **動態圖表**：根據資料庫資料動態繪製統計圖表
- **資料庫設計**：投票表、選項表、投票記錄表

## 評分標準

- 功能完整性 (40%)
- 圖表美觀度 (30%)
- 程式碼結構 (20%)
- 創意性 (10%)
