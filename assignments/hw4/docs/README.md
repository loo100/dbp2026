# 作業 4：購物車 (Cookie 實務應用)

## 任務目標

在不強制登入的情況下，使用 Cookie 暫存使用者的購物清單。

## 功能需求

1. **加入購物車**：選擇商品並指定數量
2. **修改數量**：更新購物車中商品的數量
3. **刪除商品**：移除不需要的商品
4. **計算總金額**：自動計算購物車總金額

## 技術重點

- **Cookie 使用**：`setcookie()`, `$_COOKIE`
- **序列化**：`serialize()`, `unserialize()`
- **JavaScript 互動**：動態更新購物車
