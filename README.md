# 資料庫程式設計 2026
## 陳博智老師

課程說明

以網頁進行的資訊服務系統及手機APP是目前最廣泛使用的資訊服務系統樣態。二者大抵都是以 http 作為主要的傳輸協定。

http 傳輸協定的核心是 Request - Response 。沒有任何記憶！

我們可以想像，最初的服務就是類似維基百科的靜態網頁。送出一個網頁要求，伺服器回應一個網頁檔案(HTML)，藉由瀏覽器觀閱。

當我們需到呈現較有彈性的內容時，我們會需要撰寫程式來處理不同的需求，送出符合需求的內容。這些程式需要在伺服器端運作，然後將要呈現在瀏覽器的內容準備好，送到提出要求的瀏覽器。

為了方便這樣的應用，新的程式語言紛紛被開發出來，其中 PHP 是到目前還相當活躍的網頁程式語言，本課程就是以 PHP 作為基礎。

由於 AI 普及，要求同學手刻每一行程式碼已經是不可行了，為了與時俱進，本課程鼓勵大家多多應用 AI 協助開發，在不必熟悉指令語法的前提下，訓練自己成為可以運用工具開發網站服務系統的軟體工程師(或是系統開發師)。

為了與開發工程師的工作流程接軌，我們要求大家學習使用 GitHub, 也學習以所謂的 vibe coding 的方式進行學習。目標就是建立相信自己也可以好好使用資訊工具的信心！
