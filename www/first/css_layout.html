Flexbox 常用屬性：
• display: flex; - 啟用彈性佈局
• flex-direction: row | column; - 主軸方向
• justify-content: center | space-between | space-around; - 主軸對齐
• align-items: center | flex-start | flex-end; - 交叉軸對齐
• gap: 20px; - 項目間距

浮動右側

這是一個演示文本。當元素浮動時，其他內容會環繞它。Float 曾經是實現多列佈局的常見方法， 但現在 Flexbox 和 Grid 是更好的選擇。Float 仍然用於讓文本環繞圖像等特殊場景。

Float 使用場景：
• 文本環繞圖像
• clear: both; - 清除浮動
• overflow: auto; - 父容器包含浮動子元素

📰 Multi-Column Layout (多欄佈局)

CSS 多列佈局允許您輕鬆定義多列文本。這對於報紙或雜誌風格的佈局非常有用。 文本會自動分流到各列，並且會自動平衡內容高度。您可以設定列數、列寬、列之間的間隙和列分隔線。 當瀏覽器窗口改變大小時，列會自動調整。這是一個強大的功能，特別適合長形式的內容展示。 多列佈局使用起來非常簡單，只需要設定幾個 CSS 屬性就可以實現複雜的版面設計。

Multi-Column 屬性：
• column-count: 3; - 列數
• column-width: 200px; - 列寬
• column-gap: 30px; - 列間距
• column-rule: 2px solid #ccc; - 列分隔線
• column-span: all; - 跨越所有列
